S5key - Slideshow Launcher

Ich hab kürzlich noch gedacht, dass so etwas verdammt praktisch wäre - beim heutigen Lesen von Jonathan Snooks Blog war es dann da: S5key.

Kurz für diejenigen, die S5 nicht kennen: Es handelt sich um ein kleines aber sehr feines Präsentations-System auf Basis von XHTML, CSS und JavaScript - quasi ein Webstandard-Powerpoint ;)

S5key ist nun quasi eine Erweiterung dafür: Ein durch Apple Keynote inspirierter Launcher, der die Präsentation in einem seperaten Fenster öffnet (wandert für die Zuschauer auf den Beamer) und im anderen Fenster eine Übersichtsseite für den Sprecher anzeigt. Auf dieser Kontrollseite sieht der Sprecher die aktuelle und nächste Folie, Notizen zur angezeigten Folie und eine Uhr mit Timer.

Super praktisch und dabei so einfach - wer S5 mag, wird S5key lieben :)
