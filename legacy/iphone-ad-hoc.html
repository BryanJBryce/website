iPhone: Ad Hoc Provisionierung

Testversionen

Um Ihnen Testversionen Ihrer iPhone App zur Verfügung zu stellen sind zwei Schritte notwendig: Zunächst müssen Sie mir den Identifier Ihres Testgeräts (iPhone oder iPod Touch) mitteilen, damit die Testversion Ihrer Applikation für Ihr Gerät zertifiziert werden kann. Anschließend bekommen Sie die Testversion von mir zugeschickt und können diese mittels iTunes auf Ihrem Testgerät installieren.

Ermitteln des Identifiers (bzw. UUID)

Jedem iPhone ist eine Kennung zugeordnet, die für das physische Gerät einmalig ist. Diese Kennung ist anonym und  wird gemeinhin als die “UUID” (Universal Unique Identifier) bezeichnet. Sie können die UUID Ihres Testgeräts mit Hilfe der folgenden Anweisung finden - alternativ können Sie auch die freie App AdHoc Helper App aus dem AppStore verwenden, um mir Ihre UUID per E-Mail zu schicken.

  1. Starten Sie iTunes und schließen Sie Ihr Testgerät an
  2. Auf der Seite “Übersicht” für Ihr Testgerät, klicken Sie auf das Wort “Seriennummer”:
  3. Die Seriennummer sollte sich ändern und das Wort “Identifizierung”, gefolgt von einer langen Reihe von Zahlen und Ziffern erscheinen:
  4. Wenn Sie die Tastenkombination STRG + C (Windows) oder Command / Apfeltaste + C (Mac) betätigen, so wird die UUID in die Zwischenablage kopiert und kann dann in eine E-Mail eingefügt werden.

Installation der Testversion

Sie bekommen von mir eine E-Mail die ein ZIP-Archiv mit zwei Dateien enthält: Die Datei dbloete_Ad_Hoc.mobileprovision dient als Grundlage zur Installation der Testversion, da sie die nötige Zertifizierung enthält. Die Datei IhreApp.app ist die Anwendung.

Beide Dateien lassen sich über iTunes auf Ihrem Testgerät installieren:

  1. Öffnen Sie iTunes
  2. Ziehen Sie die beiden Dateien in den Bereich “Programme” in iTunes.
  3. Sollten Sie die Anwendung vorher schon einmal installiert haben, antworten Sie auf die Frage, ob die Applikation ersetzt werden soll, mit Ersetzen (Ihre Daten bleiben erhalten)
  4. Synchronisieren Sie Ihr Testgerät und starten Sie die neu installierte Testversion Ihrer Anwendung
